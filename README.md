# MyGlobe Project

## Installation

### Clone repo

``` bash
# clone the repo
$ git clone https://github.com/axelfroehlicher/myglobe-front.git

# go into app's directory
$ cd my-project

# install app's dependencies
$ npm install
```

## Usage

``` bash
# serve with hot reload at localhost:4200.
$ ng serve

# build for production with minification
$ ng build
```
